# Smart Campus Management System

A full-stack web-based system designed to digitize and automate campus
operations including student management, faculty records, attendance,
and course administration.

## Modules
- Student Module
- Faculty Module
- Admin Dashboard
- Attendance Management
- Course Allocation

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MySQL

## System Architecture
The system follows a layered architecture consisting of presentation,
business logic, and data layers.

## Future Scope
- Mobile application
- Biometric attendance
- AI-based performance analysis
